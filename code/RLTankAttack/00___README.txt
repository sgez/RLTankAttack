Description:
============
2-player Python game consisting of two opponents trying to submerge their opponent in water. Modes: Human-vs-Human, Human-vs-Computer, Computer-vs-Computer. The purpose of this game is to study Learning mechanisms and their use in new games quality assurance.

Requires:
=========
Read README-Python.txt

Test the game:
==============
Run file RunGame.py: python RunGame.py
